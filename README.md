# UserAppThis is a repo for managing machine readable artifacts for the [UserApp APIs](https://www.userapp.io/). The goal is to maintain and evolve an OpenAPI, Postman Collection, and APIs.json index for the [UserApp APIs](https://www.userapp.io/).They are a constant work in progress, and will be updated on a regular basis when new information is added, or more of the specifications are supported.Feel free to submit a pull request to evolve any of the artifacts, and leave GitHub issues if you have any questions, comments, or feedback.- [Website](https://www.userapp.io/)- [Developer](https://www.userapp.io/)- [Twitter](https://twitter.com/userapp_io)- [GitHub](https://github.com/userapp-io)The [APIs.json](https://github.com/api-evangelist/userapp/blob/master/apis.json) index provides links to all the relevant links for the UserApp APIs, including the OpenAPI and Postman if they are present.
